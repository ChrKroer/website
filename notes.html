Latex

Decrease space usage for long subscripts: use \mathrlap (and \includepackage{mathtools})

Latexmk continuously compile on save: latexmk -pvc file.tex

Embed fonts in pdf the hard way, using ghostscript: gs -dSAFER -dNOPLATFONTS -dNOPAUSE -dBATCH -sDEVICE=pdfwrite -sPAPERSIZE=letter -dCompatibilityLevel=1.4 -dPDFSETTINGS=/printer -dCompatibilityLevel=1.4 -dMaxSubsetPct=100 -dSubsetFonts=true -dEmbedAllFonts=true -dUseCIEColor -sOutputFile=newFilename -f initialPdfName

Authoryear style citation with brackets: tex.stackexchange, also see IJCAI15 paper.

Biblatex remove bib fields:: \AtEveryBibItem{ \clearfield{pages}} tex.stackexchange.

Latex Emacs color bleed: tex.stackexchange, TLDR: try inserting %$ at EOL.

Emacs

Edit remote file: C-x C-f /host:path_to_file (alternatively /user@host:path_to_file)

Set mark: 'C-SPC' Go to mark: 'C-u C-SPC'

Start Eclim: /Users/ckroer/install/eclipse/Eclipse\ 2.app/Contents/Eclipse/eclimd

R language

Load CSV files: matrix_name <- as.matrix(read.csv(file="~/path/to/file", sep=",", header=TRUE))

Count frequencies of elements in list: t=table(l), where l is the list (this can run into floating point issues since numbers are handled by conversion to string).

Install package: install.packages("name")

Include package: library("name")
